Die **XML Diff Language** (auch **Microsoft XML Diff** oder **Diffgram**) wurde von Microsoft zur Angabe von Änderungen an [XML](xml)-Dokumenten entwickelt. Sie ist inzwischen durch [XML Patch](xml-patch) abgelöst worden.
